# 智能导购助手

一个基于大语言模型的真正智能购物助手，能够：

- 自然理解用户需求（意图识别、实体抽取）
- 智能推荐商品并生成个性化推荐理由
- 多轮对话，记住用户偏好
- 商品对比、详情查询
- 友好的对话式交互

## 核心能力

### 1. 需求理解
- 自然的对话式需求收集
- 识别商品类别、预算、品牌偏好、使用场景
- 主动追问缺失信息

### 2. 智能推荐
- 基于需求匹配商品
- 生成个性化推荐理由
- 支持多目标优化（性价比、性能、品质）

### 3. 商品对比
- 横向对比多款商品
- 突出各自优缺点
- 给出购买建议

### 4. 售后服务
- 订单查询
- 退换货政策咨询
- 物流查询

## 工作流程

```
用户输入 → LLM意图分析 → 工具调用 → 商品检索 → LLM生成回复 → 用户
                ↑                                           ↓
                └────────────── 对话历史 ─────────────────────┘
```

## 技术特点

- **LLM原生**: 所有NLP任务由LLM完成，无需规则引擎
- **Tool Calling**: 函数调用实现精准的商品检索
- **上下文记忆**: 完整的对话历史管理
- **商品知识库**: 向量化商品信息支持语义搜索
