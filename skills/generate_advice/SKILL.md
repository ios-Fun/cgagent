# generate_advice Skill

根据解析的体检数据和风险评估结果，生成个性化的健康建议。

## 功能

1. **基于异常指标生成具体建议** - 针对每个异常指标给出相应的饮食、运动、生活建议
2. **基于风险等级生成总体建议** - 根据总体风险等级给出优先级建议
3. **生成复查计划** - 根据风险等级建议合适的复查时间
4. **自然语言包装** - 用专业、鼓励的语气生成完整建议

## 输入

依赖以下步骤的输出：
- `parse_report` 的结构化输出
- `assess_risk` 的风险评估结果

## 输出

```json
{
  "structured": {
    "advice_items": [
      {"category": "饮食", "priority": "重要", "content": "..."},
      {"category": "运动", "priority": "建议", "content": "..."}
    ],
    "followup_plan": {
      "timing": "1-3个月内",
      "items": ["重点指标复查"],
      "note": "改善生活方式后复查"
    }
  },
  "text": "完整的健康建议文本..."
}
```

## 执行模式

使用 **executor.py** 混合执行模式：
- 规则引擎生成建议条目和复查计划（确定性）
- LLM 生成自然语言包装（灵活表达）
